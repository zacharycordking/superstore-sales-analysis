# Sales Performance & Revenue Analysis

## Project Overview

This project analyzes sales performance, profitability, customer segmentation, and regional trends using the Sample Superstore dataset. The objective was to identify key business insights and create a professional business intelligence dashboard using SQL, Python, and Tableau.

The project demonstrates a complete analytics workflow including:

* Data analysis with SQL
* Exploratory data analysis using Python
* Data visualization and dashboard creation in Tableau
* Business insight generation and executive reporting

# Key Insights

* High discounts negatively impacted profitability.
* Certain products generated strong revenue but weak profit margins.
* Technology and Office Supplies categories performed strongly overall.
* Consumer customers generated the highest revenue.
* Regional profitability varied significantly across the dataset.


# Business Recommendations

* Reduce excessive discounting on low-margin products.
* Focus marketing and sales efforts on high-performing categories.
* Investigate underperforming regions and product lines.
* Prioritize customer segments with stronger profit contribution.
* Continue monitoring regional sales and profitability trends.
